Coursework

What I actually worked on.

I'm grouping this by subject instead of by semester, since that's a more honest picture of what I can actually do. Three of these ended up turning into the calculators on the tools page, I'll point those out as they come up.

Databases & data

The course that fed directly into one of my tools was Big Data Mining with Python Integration. We covered link analysis (PageRank, TrustRank, HITS, spam farm detection), frequent itemset mining (A-Priori, PCY, FP-Growth), and a recommendation systems case study using Spotify data. My final project was a matrix factorization paper applying those ideas to real recommendation problems, and the link analysis assignment is basically the same logic behind the PageRank simulator on this site.

Outside of that, most of my database work has been hands-on SQL across a handful of real schema sets, StayWell, ConstructCo, SaleCo, LargeCo, and FACT, plus building the KimTay and StayWell databases from scratch in MySQL labs.

Programming

Java was where I picked up the object-oriented fundamentals: streams, generics, inheritance, and the Comparable interface for sorting custom objects. The bigger project here was my CIS capstone, a flashcard study app for a fictional company called LearnEdge Solutions LLC, where I produced a full deliverable set: a Statement of Work, a process flowchart, an interactive mockup, a Standard Operating Procedure, a Data Transfer Specification, and a full Technical Specification document. It was less about the code and more about learning what it actually takes to hand a product off to a real team. And the HTML/CSS fundamentals from this track are literally what this site is built with.

Mathematics

My calculus and optimization coursework covered derivatives, optimization problems, and root-finding methods, including Newton's Method, which became the second tool on this site. Seeing the tangent-line approximation actually converge on a canvas, instead of just on paper, is honestly part of why I wanted to build it in the first place. Beyond that, linear algebra and statistics have supported both my CIS coursework (matrix factorization, data analysis) and my ongoing actuarial exam prep. I'm currently an SOA registered candidate studying for Exam FM.

Finance & accounting

Managerial and cost accounting covered variance analysis, activity-based costing, flexible budgeting, and process costing using both weighted-average and FIFO methods. Combined with independent study on time value of money for my actuarial exam, this is where the TVM calculator came from, that formula shows up constantly once you start looking for it. Microeconomics rounded this out with marginal analysis, optimization at the margin, monopoly theory, and externalities, which is honestly the same cost-benefit thinking that shows up in the TVM tool and in how I approach tradeoffs generally.

Systems & architecture

Computer architecture and assembly language took me through Boolean algebra, the MARIE architecture, and some independent research into ARM architecture, the layer underneath the code most people only ever see running.
